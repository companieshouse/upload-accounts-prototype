Use this service to upload and file your iXBRL package accounts with Companies House.

Your accounts files must be packaged using a zip file format. This is a single file that has one or more compressed files within it. Read the Technical Interface Specification to find out how to structure your package accounts in a zip file.

You can upload the following package accounts:

  • UKSEF
  • Welsh/English
  • Community Interest Company
  • Overseas
  • Audit Exempt Subsidiary
  • Filing Exempt Subsidiary
  • Limited Partnership
  • Group Package (s400/s401)

For more information on taxonomies and guidance for filers and developers, see the Financial Reporting Council (FRC) website.
