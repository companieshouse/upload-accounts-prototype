Use this service to file your iXBRL package accounts with Companies House.

What package accounts can I file?

You can use this service to file:

  • UKSEF
  • Welsh/English
  • Community Interest Company (CIC)
  • overseas
  • audit exempt subsidiary
  • filing exempt subsidiary
  • limited partnership
  • group package (s400/s401)

You'll need:

  • to sign in or create a Companies House account
  • the company name or number
  • the company authentication code
  • the iXBRL accounts file, packaged using a zip file format
  • any related forms (if relevant)
  • to pay the £XX fee using a credit or debit card
