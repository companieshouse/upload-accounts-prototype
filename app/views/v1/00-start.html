Use this service to file your iXBRL package accounts with Companies House.

You can only file package accounts for:

  • Community Interest Companies (CIC) - there is a £15 fee to file
  • overseas companies - there is a £33 fee to file
  • audit exempt subsidiary accounts
  • dormant exempt subsidiary accounts
  • limited partnership accounts
  • UKSEF accounts for listed companies
  • group package accounts - section 400, parent incorporated under UK law
  • group package accounts - section 401, parent incorporated under non-UK law
  • Welsh accounts with an English translation

You must package your accounts file using a zip file format. Read the Technical Interface Specification to find out how to structure your file.

You'll need:

  • to sign in or create a Companies House account
  • the company name or number
  • the company authentication code
  • to pay the fee using a credit or debit card, if relevant

This service will time out if you do not use it for 60 minutes. Your answers will not be saved.
